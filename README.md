# Gladia Async Transcription

## Step-by-step Demo

## What it does
Async speech-to-text: upload audio to Gladia, get an `audio_url`, create a transcription job, and get a transcript with speaker diarization.

---

## Setup
- Get API key at https://app.gladia.io/
- Upload audio to Gladia → receive `audio_url`
- Call async endpoint (https://api.gladia.io/v2/pre-recorded) → receive `result_url`
- Poll or use webhook
